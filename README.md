# PVS — Personal site

Personal site and blog of Vitalii Pravdych — Tech Lead, AI & R&D. Built with Astro, deployed as a static bundle to GitHub Pages.

- **Quickstart (dev / publish / deploy):** [specs/001-portfolio-blog-site/quickstart.md](specs/001-portfolio-blog-site/quickstart.md)
- **Architecture & decisions:** [specs/001-portfolio-blog-site/plan.md](specs/001-portfolio-blog-site/plan.md), [research.md](specs/001-portfolio-blog-site/research.md)
- **Hard rules:** [.specify/memory/constitution.md](.specify/memory/constitution.md)

> The deploy assumes the GitHub repo is named `PVS` (so `base: '/PVS/'` in `astro.config.mjs` and the live URL is `https://<user>.github.io/PVS/`). If the repo name differs, edit `site` and `base` in `astro.config.mjs` and re-deploy. See [specs/001-portfolio-blog-site/research.md](specs/001-portfolio-blog-site/research.md) — *Open decision: `base: '/PVS/'`*.
